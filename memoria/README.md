# Memoria Clasificada

Colección de archivos markdown del servidor Nexus, clasificados por categoría.

## Estructura

```
memoria/
├── skills/          # Skills de OpenCode/Claude
├── nexus-os/        # Planes y guías de Nexus OS
├── resumenes/       # Resúmenes de conversaciones
├── otros/          # Varios

```
